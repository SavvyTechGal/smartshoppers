Learn the Lingo

Here is our short and simple guide to some of the most common words used when shopping for computers and laptops!

Search Term

Central Processing Unit (CPU)

  • A central processing unit, in short, is responsible for the computing power and handles data processing operations.
  • A computer with a good CPU will operate faster and can handle more intensive applications.

Processor Speed

  • Measures how fast or slow your computer recieves information and processes it
  • This is determined by both processor cores and clock speed.

Clock Speed

  • Number of calculations per second
  • Clock speed is measured in Gigahertz (GHz).
  • 1 GHz = 1 billion calculations per second.
  • The higher the clock speed, the faster your computer completes tasks.

Processor Cores

  • These are individual processing units within the computer's CPU that take intructions for a single task.
  • Having multiple cores allow the computer to multi-task.

Random Access Memory (RAM)

  • This is where your computer temporarily stores information/applications and allows it to complete everyday tasks.
  • RAM provides fast access which allows the user to switch between many apps quickly.
  • The more applications users need to run, the more RAM is needed.
  • Generally the more complex programs, such as games or design software, will take up the most RAM.

Storage

Solid State Drive (SSD)

  • Uses flash-based memory
  • No moving parts, which allows for more compactability and faster access
  • Faster operations (booting PC, launching applications)
  • Recommended for optimizing performance
  • Significant performance improvement over HDD, but much more expensive.

Hard Disk Drive (HDD)

  • spinning disk that stores data and the information is accessed using a reading arm that scans the surface
  • Cheaper medium of storage, but is much slower
  • Recommended for data that does not need to be accessed frequently or quickly.

Graphics Processing Unit (GPU)

  • processor specifically for rendering graphics and video
  • often used for creative production, machine learning/AI
  • Recommended for advanced visual effects and gaming

Integrated GPU

  • when the GPU is integrated with the CPU
  • allows for more light and compact systems and lower costs

Discrete GPU

  • when the GPU is separate from the CPU
  • recommended for more intensive applications, like gaming and video creation

Display Resolution

  • amount of pixels in the monitor to display an image
  • measured in height and width
  • The higher the resolution, the more pixels there are and the more high-quality sharp image you will get.
  • Most common resolution: 1920 x 1080 Full HD

Refresh Rate

  • How many times the monitor refreshes the image within one second.
  • The standard rate is 60Hz, in which the screen refreshes 60 times in one second. Rates can go up to 144Hz, 240Hz, etc.
  • Higher refresh rates may be desirable for smoother gaming experiences.
  • To get the full benefit of fast refresh rates, you may want to consider getting a CPU or GPU that is fast enough to handle instructions.

Display Panel

  • Determines how information is grouped together to be displayed to the user
  • Affects how images are displayed on a monitor screen

Panel Types

TN (Twisted Nematic) Panel

  • Cheapest and most common type
  • (+) Fast responsiveness
  • (-) Poor color quality and viewing angles. Not recommended for professional color work.

IPS (In-Plane Switching) Panel

  • Most expensive type
  • (+) Best viewing angles and color accuracy
  • (-) Poor contrast (not good with dark images).
  • Most recommended for professional work.

VA (Vertical Alignment) Panel

  • Considered the middle standard
  • (+) Adequate viewing angles and color accuracy (not as good as IPS)
  • (+) Best for dark colors/scenes

Ports

  • Allow connection between computer and peripheral devices (mouse, keyboard, headphones, etc)

Universal Serial Bus (USB)

  • the most common port
  • allows connection to devices such as other computers, storage drives, phones for file transfer, charging, etc

HDMI Port

  • sends video and audio signals
  • commonly used to connect to display monitors, soundbars, game consoles, etc.

Ethernet

  • allows direct connection to wired networks
  • recommended option if wireless connection signal is poor

Audio-Jack

  • allows connection to headphones, mic and speakers
